# Wilton Paulo - DevOps & Cloud Engineering Blog

Personal blog about DevOps, Cloud, and Infrastructure. Articles on Terraform, Kubernetes, AWS, GCP, and automation.

Built with [Hugo](https://gohugo.io/) and the [Hextra](https://imfing.github.io/hextra/) theme.

## Features

- **Professional Tech Content:** Focus on DevOps, SRE, and Cloud Architecture.
- **Medium-style Design:** Clean, minimal, and focused on readability with serif fonts and centered content.
- **Bilingual Content:** Support for English (default) and Portuguese.
- **Dark Mode:** Built-in theme toggle.
- **Search:** Integrated Flexsearch for fast content discovery.

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions on every push to the `main` branch.

## Local Development

1. Install Hugo (Extended version)
2. Clone the repository with submodules:
   ```bash
   git clone --recursive https://github.com/wiltonpaulo/wiltonpaulo.github.io.git
   ```
3. Run the development server:
   ```bash
   hugo server
   ```
